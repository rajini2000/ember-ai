OLED Display Integration with Live AI Alarm Status

Connect a 1.3" SH1106 OLED display (128×64, I2C) to the FRDM-K64F on the shared I2C bus
(SDA=PTE25, SCL=PTE24, address 0x3C). Write an SH1106 driver from scratch using raw I2C
commands (no external library).

The OLED displays a real-time dashboard showing:
- PM2.5 value with a progress bar
- Composite AQI score with EPA category label (GOOD/MODERATE/UNHEALTHY/etc.)
- Temperature and humidity from the BME680
- MQ gas voltage
- Current alarm state (SAFE / DANGER) from the team’s DQN AI model via cloud API

Display updates every 2 seconds.

If AI returns alarm=ON → OLED shows flashing “⚠ DANGER” warning with inverted colors.

COMPLETION CRITERIA
✓ SH1106 OLED detected and initialized with custom driver
✓ PM2.5 value and progress bar updating every 2 seconds
✓ AQI score with EPA category label displayed
✓ Temperature, humidity, and MQ gas voltage displayed
✓ AI alarm status shown as SAFE or flashing DANGER
✓ Display persists across power cycles and WiFi reconnections

REST API Integration — K64F to Cloud via ESP32 HTTP

Program ESP32 to send sensor data from FRDM-K64F to the team's Flask REST API
(deployed on Render.com) using HTTP POST requests.

Each cycle K64F sends JSON payload with:
- PM2.5
- PM10
- Temperature
- Humidity
- Gas voltage
- Composite AQI

API returns alarm prediction from DQN model.
K64F parses JSON response and activates or deactivates the physical alarm.

Hardware → Cloud AI → Hardware Action loop.

COMPLETION CRITERIA
✓ K64F sends sensor JSON to API every cycle
✓ API returns alarm prediction (ON/OFF) and AQI estimate
✓ Physical alarm on PTA2 activates/deactivates based on response

PROJECT CONTEXT

Ember is an indoor air quality monitoring and fire detection system built on the
FRDM-K64F (ARM Cortex-M4).

Sensors used:
- PMS5003 laser particle sensor
- BME680 environmental sensor
- MQ gas detector
- ESP32 WiFi/BLE module

Features:
- Composite AQI calculation
- MicroSD logging
- Cloud API streaming
- DQN-based alarm model
- Web dashboard with digital twin

Mirac handles:
- Embedded firmware

- WiFi provisioning portal
- Fire detection algorithm
